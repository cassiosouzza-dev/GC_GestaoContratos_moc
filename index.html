GC Gestor
Soluções em Contratos Públicos
Versão 1.0

Sistema Integrado de Gestão Contratual

Software especializado para auditoria de empenhos, controle de saldos orçamentários, gestão de aditivos e conformidade administrativa. Projetado para eficiência e transparência.

Download do Executável (v1.1)
Compatibilidade: Microsoft Windows 10/11 (64-bits) • Arquitetura Portable (Sem Instalação)

Análise Preditiva e IA

Módulo integrado de inteligência artificial para consulta em linguagem natural e análise de riscos contratuais, identificando inconformidades em datas e saldos.

Integridade de Dados

Sistema de versionamento com backups automáticos locais (.bak) e funcionalidade de reversão de operações críticas (Rollback/Undo), garantindo segurança da informação.

Relatórios Gerenciais

Emissão de extratos detalhados em formato HTML/PDF, prontos para compor processos administrativos e prestações de contas oficiais.

Procedimentos de Instalação e Acesso

1

Download e Extração

O sistema utiliza arquitetura Portable para facilitar o uso sem privilégios administrativos. Baixe o arquivo comprimido (ZIP) e extraia todo o conteúdo para uma pasta local segura.

2

Configuração de Ambiente

Certifique-se de que o arquivo chave_api.txt (contendo a credencial da API Gemini) esteja presente no diretório raiz para habilitar os módulos de inteligência.

3

Inicialização e Credenciais

Execute o arquivo GC_Gestor_v1.exe. No primeiro acesso, selecione a opção "Criar Nova Conta" para definir as credenciais de Administrador do sistema.

4

Operação Inicial

Após a autenticação, acesse o menu Cadastros > Gerenciar Prestadores para iniciar o povoamento da base de dados.
